## ПРОМПТ

Создай одностраничный сайт-визитку для [ПРОФЕССИЯ] [ИМЯ ФАМИЛИЯ].

### Общие требования:
- Vanilla HTML + CSS + JavaScript (без фреймворков)
- Темная цветовая схема (#08080C фон, #8B7FFF акцент, фиолетово-голубой градиент)
- Шрифты: Space Grotesk (заголовки) + Inter (текст), подключить через Google Fonts
- Дизайн уровня Awwwards: glassmorphism, плавные анимации, микро-интеракции
- Адаптивный дизайн (desktop, tablet, mobile)
- Кастомный курсор с blend-mode (скрывается на мобильных)

### Структура файлов:
```
/index.html
/css/style.css
/js/three-scene.js    — 3D анимация
/js/animations.js     — GSAP ScrollTrigger
/js/ai-demo.js        — интерактивный AI-демо (если нужен)
/assets/images/        — фото и изображения
```

### Hero секция (первый экран):
- **Layout**: двухколоночный flex. Слева — текст, кнопки, статистика. Справа — фото специалиста
- **Фото**: абсолютно позиционировано справа (`position: absolute; right: -100px; top: 10%; width: 65%`), PNG с прозрачным фоном, с drop-shadow и фиолетовым glow снизу
- **Текст слева**: badge-плашка сверху, крупный заголовок (clamp 48px-88px), подзаголовок, две кнопки (gradient primary + ghost), плашка статистики (3 числа с разделителями)
- **Фон**: Three.js canvas с нейросетевой анимацией частиц (180 частиц, линии-связи между ближними, без центральной сферы). Частицы реагируют на движение мыши
- **Z-index**: canvas=1, portrait=2, text=3 (текст поверх фото)
- **Preloader**: орбитальные кольца + инициалы, скрывается через 2.5с с 4с fallback

### Навигация:

### Анимации (GSAP + ScrollTrigger):
- `[data-animate]` элементы: opacity 0 → 1, translateY/X → 0 при скролле в viewport
- Hero pin: hero закреплен пока About не наедет сверху
- `data-delay` на элементах для stagger-эффекта (НЕ умножать на 1000 — значения уже в секундах!)
- Счетчики чисел (countUp) в hero-stats
- Magnetic-эффект на кнопках при наведении
- Preloader: обязательный safety-net fallback через setTimeout

### Критические баги которых нужно избежать:
1. **НЕ** ставить `scroll-behavior: smooth` на html — конфликтует с GSAP ScrollTrigger pin
2. **НЕ** умножать data-delay дважды (значение уже в секундах: 0.1, 0.2, 0.3)
3. **Обязательно** safety-net для preloader (setTimeout 4с force-remove)
4. **Обязательно** `overflow: hidden` на hero чтобы фото не создавало горизонтальный скролл
5. Hero контент z-index=3 ВЫШЕ чем фото z-index=2 чтобы текст был кликабельным поверх фото
6. На мобильных: скрыть кастомный курсор (`display: none !important`), вернуть `cursor: auto`
